本文件夹作用：
前端开发源文件夹，使用vue框架，实现前端页面的构建和功能实现   开发工具(VSCode)

各文件夹作用：
build:                    vue打包工具
config:                  vue配置文件       
dist:                       vue打包完成后的文件夹，需要放在服务器运行
node_modules:     vue运行依赖文件

src:                        前端源代码
         assets:   存放图片、视频和css文件
         components: 页面用到的组件
         router:    vue路由文件
         store: 配置存放session缓存的文件
         views:  web页面的vue文件

static:                    配置文件
